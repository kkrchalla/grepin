Tips about the search options
  • Type of SearchBox
    • Give the type of the searchbox you would like to build.
    • A simple searchbox would be the most common searchbox with a text box for query.
    • A categorized searchbox will let your users to search specific areas of your web site.
    • A text link searchbox is a predefined search with just a link.
    • If you do not give any value, the default would be a simple searchbox.
  • User ID
    • You should give your account number. If you have logged in, the account number will be automatically filled.
  • Query
    • If you are building a text link searchbox, the query in this field will be used to search your web site.
  • Page number
    • This would be the page number of the search results. If no value is supplied, the first page will be accessed.
  • Include parameters
    • You can limit your search to a specific directory or directories or certain number of pages by giving a value in this option.
    • Example: If you give a value of '/books/', the search will only be performed among the pages that have '/books/' in their url path.
    • Example: If you give a value of '/books/|/videos/', the search will only be performed among the pages that have either '/books/' or '/videos/' in their url path.
    • Example: If you give a value of 'book', the search will only be performed among the pages that have 'book' in their url path. This word can occur anywhere in the url.
  • Exclude parameters
    • You can exclude certain pages or directories from being searched.
    • Example: If you give a value of '/books/', all the pages whose url have the word '/books/' will be excluded from the search.
    • Example: If you give a value of 'book', all the pages that have 'book' in their url path will be excluded. This word can occur anywhere in the url.
  • Priority
    • Use this option to give high priority to the new documents over the old ones.
    • This field can hold a number between 0 and 10. The greater the number the higher the priority to the new documents.
    • If no value is given, all documents are given equal priority.
  • Sort field
    • You can sort the results by rank, title, last modification date, and file size.
    • A value of 'rank' will sort the results by rank.
    • A value of 'title' will sort the results by title.
    • A value of 'date' will sort the results by last modification date.
    • A value of 'size' will sort the results by file size.
    • If you do not give any value, the results will be sorted by rank.
  • Results per page
    • You can control the number of search results per page by giving a value in the option.
    • If no value is given, 10 results per page will be displayed.
    • The valid values are 5, 10, 15, 20, 25, 30, 35, 40, 45, 50, 75, or 100.
  • Display snippets
    • If you want to display the snippets (containing the search terms) from the contents of your page in the search results, give a value of '1' in this option.
  • Mode
    • The values can be 'all' or 'any'. If no value is given, 'any' will be assumed.
    • A value of 'all' will return a page in the results only if all the search terms are present in the page.
    • A value of 'any' will return a page in the results if any of the search terms is present in the page.
  • Location
    • The values can be 'us' or 'uk'. If no value is given, 'us' will be assumed.
    • If 'us' is specified, the products from Amazon.com will be advertised.
    • If 'uk' is specified, the products from Amazon.co.uk will be advertised.
  • Ads category
    • You can limit the products in the ads to a specific category by giving a valid amazon category here.
    • For valid categories, click here.
    • If nothing is specified here, the default is to advertise products from all categories.
  • Your Amazon associate ID
    • You should give your amazon associate id here.
    • If locale is 'us', you should give your Amazon.com's associate id.
    • If locale is 'uk', you should give your Amazon.co.uk's associate id.
  • Border color
    • Border color of the advertising bars.
    • If nothing is specified, the default color 'orange' will be used.
    • This can be literal or a hex value of the color.
  • Footer text color
    • Footer text color of the advertising bars.
    • If nothing is specified, the default color 'white' will be used.
    • This can be literal or a hex value of the color.
  • Source
    • This option is for your tracking purposes. If you have search box in multiple pages of your web site, you can use the value of source to track all the searches. The value of the source option will show up in the detailed search activity report.
